
# KOCAMP

## 개요
주제 선정을 위해 웹과 앱들을 찾아보던 중 코로나 이후 캠핑하는 사람들이 많아짐에 따라 캠핑장 정보와 해당 캠핑장의 리뷰를 쉽게 볼 수 있는 웹앱이 있으면 좋을 것 같아 주제로 정하게 되었습니다.
